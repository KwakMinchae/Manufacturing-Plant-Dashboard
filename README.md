Dashboard based on raw data from Current and Vibration Sensor installed to machines in a manufacturing plant.

this is a fully deployed and an interactive web dashboard. Used python (streamlit + Plotly). Deployed live on Streamlit cloud.

Click 'Yes, get this app back up!' to view the dashbaord: https://plant-dashboard-k9yiwnxyt8wymhmqbgrkuk.streamlit.app/

The dashboard contains the following info.

PDF export generating 1 page summary report of the day
OEE (Overall Equipment Effectiveness) cards
KPI cards
Scatter plot, box plot, etc,.
